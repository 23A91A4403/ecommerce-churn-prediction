# Project Scope

## In Scope
- Customer-level churn prediction
- RFM (Recency, Frequency, Monetary) analysis
- Feature engineering from transaction data
- Machine learning model development and evaluation
- Batch and single customer churn predictions

## Out of Scope
- Real-time prediction systems
- Product recommendation engines
- Inventory optimization
- Paid tools or APIs

## Timeline
- Total duration: **8 weeks**

## Constraints
- Use of **free and open-source tools only**
- Public deployment using free platforms
- Limited to historical transaction data
